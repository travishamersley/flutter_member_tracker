# membership_tracker

Membership Tracker is a Flutter application that helps you track members and attendance at a small club that has a simple pay per lesson billing structure.

## Design goals
Create an App that is primarly targeting the web secured via Google accounts. There will be no server, Google Drive and Google Sheets will be used to store data. The app will be used by multiple instructors to track attendance and payments for their own groups of students. Instructors will be able to share their data with other instructors by sharing their Google Sheets.  
Where possible use core Flutter packages and avoid adding unnecessary dependencies.

## Features

- Manage members (add, edit, delete)
- Track attendance (mark members as present or absent)
- Start and end lessons
- Track payments and balances
- Support grouping members into a family
- Support payment balances and credits to apply to all family members
